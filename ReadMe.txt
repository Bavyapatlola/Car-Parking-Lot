How to run the application?

1) Open project in IntelliJ IDE
    Importing java project : https://bit.ly/2WfNvU5
2) JAVAFX Library
    How to download: https://gluonhq.com/products/javafx/
3) VM options
    How to add edit configaration for main(): https://openjfx.io/openjfx-docs/
4) Mysql Connector
    How to download mysql connector: https://dev.mysql.com/downloads/connector/j/5.1.html
5) How to connect to database.
    - download mysql community : https://dev.mysql.com/downloads/installer/
    - Open Connection.java file and give the following values based on the local databse.
    - PASSWORD = "<Value>";
	- USERNAME = "<Value>";
    - JDBC_DRIVER = "com.mysql.jdbc.Driver";
	- String URL = "jdbc:mysql://<Local host>/dbfp?useSSL=false";
